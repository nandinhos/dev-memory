# Neo-Brutalist Design Kit v1.0.0

Bold, high-contrast design kit with thick borders, offset shadows, Oswald typography, and a vibrant palette on cream background.

## Design Tokens

- **Primary accent**: `#FDE047` (Yellow)
- **Background**: `#F0EAD6` (Cream)
- **Border**: `#000000` (Black, 2px)
- **Shadow**: `4px 4px 0 #000` (hard offset)
- **Font heading**: Oswald
- **Font body**: Space Mono
